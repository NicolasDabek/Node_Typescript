Pour démarrer un projet sequelize:
sequelize init --force

version javascript:
npm i -g sequelize sequelize-auto mysql2
sequelize-auto -h localhost -d db_name -u root -p 3306 --dialect mysql
(si la base de données possède un mot de passe: -x motdepasse)

version typescript:
npm i -g sequelize sequelize-auto mysql2
sequelize-auto -h localhost -d db_name -u root -o ./src/models -p 3306 --dialect mysql -l ts
(si la base de données possède un mot de passe: -x motdepasse)

(Relancer la commande sequelize-auto pour mettre à jour les models si il y a eu un changement en base de données)
